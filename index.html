Sobre mim

Dados como vantagem competitiva.

Sou um engenheiro focado em construir a infraestrutura que transforma dados brutos em decisões inteligentes. Minha atuação abrange desde a extração assíncrona de dados até a modelagem dimensional em warehouses modernos.

Tenho experiência prática com pipelines ETL completos, design de APIs RESTful com FastAPI e orquestração de transformações com dbt. Trabalho com dados de comércio exterior, integrando APIs públicas, armazenamento em cloud e visualização analítica.

Valorizo código limpo, documentação clara e soluções que escalam — desde um scraper assíncrono eficiente até um data warehouse dimensional no Snowflake.

ComexData ETL

Pipeline ETL completo para dados de comércio exterior brasileiro. Extrai dados da API Comex Stat, carrega no Snowflake e aplica transformações dimensionais via dbt — gerando tabelas de fatos e dimensões prontas para análise com BI.

Snowflake dbt Python Parquet

Inventory CRUD

Aplicação web full-stack para gestão de inventário. Backend com FastAPI expondo endpoints RESTful, frontend em Streamlit, banco de dados PostgreSQL e toda a stack containerizada com Docker Compose — incluindo rastreamento de histórico de movimentação de produtos.

FastAPI Streamlit PostgreSQL Docker

Korean Trade Scraper

Scraper assíncrono de alta performance para coleta de estatísticas de comércio internacional entre Coreia do Sul e países parceiros. Utiliza API oculta descoberta via inspeção de página, com aiohttp e asyncio para requisições paralelas eficientes — exportando os dados estruturados em CSV.

Python aiohttp asyncio Pandas
